What is Outreachy, anyway? Outreachy is a scholarship program for underrepresented groups (LGBTQ+ people, PoC and women) in open source software development. It is organised by nice people of GNOME Foundation, Software Freedom Conservancy and Red Hat. Many-many cool companies support the program and work with the students.
